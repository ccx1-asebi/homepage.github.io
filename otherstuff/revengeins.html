a pretty scuffed tutorial on how to install Revenge (vendetta but not dead) on your phone since i don't wanna explain it to all of my friends individually

you will first need to install vendetta manager from

the official vendetta github repository

install the apk, you may need to enable installs from unknown sources so you are gonna have to figure out how to do that on your own

once it is installed head over to settings > about and then tap on the apps version number till something happens

- -

after something happened you should have developer settings enabled in the settings page

in dev settings you will need to change the Xposed module location to

/storage/emulated/0/Download/app-release.apk

and then you will need to download the Xposed module from

Xposed module direct download link

do not install the module once it is installed

just let it download and then leave it alone

once everything above has been done go back to the main screen and install vendetta (revenge) and wait for it to finish everything. make sure you press install once it prompts you and not cancel by accident

once it installs, launch vendetta and head over to settings

if settings contains a revenge section then you have succesfully installed revenge!
